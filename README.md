# Local Med-Voice Agent 🏥🔊
**Problem:** Visually impaired patients cannot read complex medical reports, and uploading them to cloud AI violates patient privacy.
**Solution:** A 100% offline Python tool that uses local Text-to-Speech (and eventually Local LLMs) to read and summarize medical documents without internet access.
**Why I need Claude OSS:** I am building privacy-first healthcare tools. I need Claude's reasoning to help me integrate local OCR (Optical Character Recognition) to scan physical reports securely on consumer hardware.
